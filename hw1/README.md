# hw1 : Hierarchical model

==========================

## Recommended Environment

 - Linux
 - OpenGL >= 3.0
 - C++ >= 6.3.1
 - CMake >= 3.7.2

## Execution
 - `mkdir build`
 - `cd build`
 - `cmake ..`
 - `make`
 - `cd ../hw1/` (Directory should be correct, since it loads `obj` and shader files in relative path)
 - `./hw1`
